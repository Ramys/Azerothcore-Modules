### Description:
Auto resurrection at death only for GM

### Configuration:
```ini
###################################################################################################
#	mod-auto-revive
#
#    AutoRevive.Enable 
#        Description: Enable auto revive before die, only GM account.
#        Default: 1
#
#    AutoRevive.ZoneID 
#        Description: Enable in a certain area, if it is 0, it means everywhere.
#        Default: 0
#

AutoRevive.Enable = 1
AutoRevive.ZoneID = 0
###################################################################################################
```

### How to use:
Change config and try to die.
