### Описание:
Добавление новой кастомной команды `.buff`

### Конфигурация:
```ini
###################################################################################################
#	mod-buff-command
#
#    Buff.Command.Enable
#        Описание: Включить команду .buff.
#        По умолчанию: 1
#

Buff.Command.Enable = 1
###################################################################################################
```

### Как использовать:
База данных `world`.`player_buff`:

- `SpellID` - Номер спелла (обязательно)
- `Comment` - Ваш комментарий (не обязательно)
