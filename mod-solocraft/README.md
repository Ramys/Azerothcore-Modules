# mod-solocraft

#### A module for AzerothCore by [StygianTheBest](https://youtu.be/fERCwTTOU3M?t=126) ####
------------------------------------------------------------------------------------------------------------------
- Adjusts player stats for raids based on the # of players in the group
- Config: Difficulty settings for each instance type
