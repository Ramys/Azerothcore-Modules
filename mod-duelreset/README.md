# Duel Reset #

### Description ###
------------------------------------------------------------------------------------------------------------------
- The original module has been modified to work with StygianCore.
- Resets player attributes after dueling.

### Data ###
------------------------------------------------------------------------------------------------------------------
- Type: Server/Player
- Config: Yes
